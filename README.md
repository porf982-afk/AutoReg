![Python](https://img.shields.io/badge/Python-3.8%2B-blue)

**AutoReg** - это инструмент для автоматизации процесса регистрации на любых веб-сайтах.  
Записывает последовательность кликов и повторяет её с подстановкой случайных данных: имени, временной почты, пароля.

## ✨ Возможности

- 🖱️ **Запись макроса** - просто кликни по полям в браузере, скрипт запомнит координаты.
- 📧 **Временная почта** - автоматическое создание одноразовых ящиков.
- 👤 **Управление именами** - "база данных" username с отслеживанием использованных.
- 🔐 **Генератор пароля** - случайные и обязательные комбинация букв, цифр и спецсимволов.
- 📊 **Статистика** - отображение прогресса и итогового результата.

## 📋 Требования

- Python 3.8 или новее
- Установленный браузер **Google Chrome**
- **ChromeDriver** в PATH или в папке со скриптом
